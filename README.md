# Deze Portfolio

This repository contains a customized Next.js portfolio homepage for showcasing selected projects with a clean, responsive layout and a lightweight modal-based project launch flow.

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Project Structure

- `src/app` - App Router entry files, metadata, and global styles.
- `src/components` - Reusable homepage sections and shared SVG icon components.
- `src/data/portfolio.ts` - Portfolio content and navigation/project configuration.

## Notes

- The homepage content is intentionally organized into reusable sections so additional projects or sections can be added without expanding `src/app/page.tsx` into a large monolith.
- Starter assets generated by the default Next.js template were removed after confirming they were no longer used by the portfolio implementation.
